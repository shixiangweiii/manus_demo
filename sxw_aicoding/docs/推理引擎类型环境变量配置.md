## 目前推理类型介绍
- simple: flat plan -> sequential execute -> reflect
- complex: v2 DAG -> parallel super-step execute -> reflect_dag
- emergent:
    - v5 emergent planning via TODO list
    - v8 goal-driven engine // ENABLE_GOAL_DRIVEN_PLANNER=true 配置环境变量时
- 详细逻辑参考源码：
  - agents.planner.PlannerAgent.classify_task
  - agents.orchestrator.OrchestratorAgent.run
## 通过 PLAN_MODE 配置指定类型
```
ENABLE_GOAL_DRIVEN_PLANNER=true
PLAN_MODE=emergent
```
详细参考 ".env"

## 完整说明

系统共有 4 种推理引擎，由两个核心环境变量组合决定执行路径：

### 核心路由变量

| 变量 | 默认值 | 作用 |
|------|--------|------|
| `PLAN_MODE` | `auto` | 决定走 simple / complex / emergent 路径 |
| `ENABLE_GOAL_DRIVEN_PLANNER` | `false` | 在 emergent 路径内决定走 v5 还是 v8 |

### 各引擎所需配置

#### 1. Simple (v1 扁平计划 — 顺序执行)

```
PLAN_MODE=simple
```

- 执行路径: `PlannerAgent.create_plan()` → `_execute_and_reflect_simple()`
- 无需其他开关，`ENABLE_GOAL_DRIVEN_PLANNER` 不生效（emergent 路径未进入）

#### 2. Complex (v2 DAG — 并行超步执行)

```
PLAN_MODE=complex
```

- 执行路径: `PlannerAgent.create_dag()` → `_execute_dag_and_reflect()`
- 可选调优: `MAX_PARALLEL_NODES=3`, `ADAPTIVE_PLANNING_ENABLED=true`, `NODE_EXECUTION_TIMEOUT=300`

#### 3. Emergent v5 (TODO 列表驱动)

```
PLAN_MODE=emergent
ENABLE_GOAL_DRIVEN_PLANNER=false
EMERGENT_PLANNING_ENABLED=true
```

- 执行路径: `EmergentPlannerAgent.execute()` — TODO 列表 + 每 TODO 内部 ReAct 循环
- 可选调优: `MAX_TODO_ITEMS=20`, `MAX_EMERGENT_OUTER_ITERATIONS=60`, `ENABLE_REACT_ENGINE_V2=false/true`(控制 v5 内部是否用统一 ReActEngine)

#### 4. Emergent v8 (目标驱动规划 — GoalDrivenPlanner)

```
PLAN_MODE=emergent
ENABLE_GOAL_DRIVEN_PLANNER=true
EMERGENT_PLANNING_ENABLED=true
```

- 执行路径: `GoalDrivenPlannerAgent.execute()` — 目标文档锚定 + 动态 TODO + 目标反思
- 可选调优: `GOAL_REANCHOR_INTERVAL=5`, `GOAL_REFLECTION_INTERVAL=1`, `MAX_GOAL_DRIVEN_ITERATIONS`, `GOAL_DRIVEN_STAGNATION_WINDOW=3`

#### 5. Auto (自动分类 — 默认模式)

```
PLAN_MODE=auto
```

- 执行路径: `PlannerAgent.classify_task()` → 规则快筛 + LLM 分类 → 自动选择 simple/complex/emergent
- emergent 子路径再由 `ENABLE_GOAL_DRIVEN_PLANNER` 决定 v5 还是 v8
- 特殊规则: `EMERGENT_PLANNING_ENABLED=false` 时，所有 emergent 分类降级为 complex

### 完整决策树

```
PLAN_MODE=simple  ──→ v1 扁平计划
PLAN_MODE=complex ──→ v2 DAG 并行
PLAN_MODE=emergent
    ├── ENABLE_GOAL_DRIVEN_PLANNER=true  ──→ v8 目标驱动
    └── ENABLE_GOAL_DRIVEN_PLANNER=false ──→ v5 TODO 列表
    └─ 特殊: EMERGENT_PLANNING_ENABLED=false → 降级为 v2 DAG
PLAN_MODE=auto
    ├── 规则/LLM 分类为 "simple"   ──→ v1
    ├── 规则/LLM 分类为 "complex"  ──→ v2
    └── 规则/LLM 分类为 "emergent"
        ├── ENABLE_GOAL_DRIVEN_PLANNER=true  ──→ v8
        └── ENABLE_GOAL_DRIVEN_PLANNER=false ──→ v5
```
